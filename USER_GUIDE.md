## 🔗 ภาพรวม: 4 Role เชื่อมกันอย่างไร

```
ผู้ขาย ──สร้างงาน──▶ ผู้ซื้อ/คนขับ ──ปิดงาน──▶ ออกบิลอัตโนมัติ ──▶ ระบบร้าน (บัญชี)
  ▲                       │                        │                      │
  │                       │                        ▼                      ▼
  └──── สิทธิ์รางวัล ◀──── ปิดงานสำเร็จ ────▶ ผู้ขายได้เงิน+สิทธิ์      GMV + ค่าคอม
                                                   │                      │
                                                   └──────▶ เจ้าของระบบ (แอดมิน) ◀┘
                                                        เห็นทุกบิล/รายได้/สิทธิ์
                                                        สุ่มรางวัล + แจ้ง LINE + ระงับบัญชี
```

**หัวใจการเชื่อม:** ทุกครั้งที่ "ปิดงาน" (ไม่ว่าจากมือถือคนขับ หรือหน้าร้าน) ระบบจะ **ออกบิลอัตโนมัติ** → ปิดงาน → ออกสิทธิ์ให้ผู้ขาย → เข้า GMV ของแอดมิน ครบในคลิกเดียว

## 👤 1. ผู้ขาย (Seller) — คนมีของเก่าจะขาย

**เข้าเดโม:** หน้า login → ปุ่ม **"ผู้ขาย"**

1. **หน้าแรก** — ดูราคาของเก่าวันนี้ (ราคากลางที่แอดมินตั้ง) + teaser รางวัล
2. กด **"สร้างรายการขายของเก่า"** (ปุ่ม + กลางล่าง)
   - เลือกของเก่า + ปริมาณ *(ข้ามได้ → ตกลงราคาหน้างาน)*
   - ปักหมุดสถานที่ (ตำแหน่งปัจจุบัน) + บ้านเลขที่ + จุดสังเกต
   - ผู้ติดต่อ + **จองวันกับผู้ซื้อ** (เลือกรอบว่าง) หรือ **โพสต์งานเปิด**
3. **รายการของฉัน** — ติดตามสถานะ: ส่งงาน → คอนเฟิร์ม → กำลังไปรับ → สำเร็จ (แจ้งเตือน LINE)
4. **แท็บรายได้** — ขายได้กี่บาท/เดือน + รวม + **สิทธิ์ลุ้นรางวัล** (100 บาท = 1 สิทธิ์) + หน้าประกาศรางวัล + ออกจากระบบ

---

## 🚚 2. ผู้ซื้อ / คนขับ (Buyer) — คนไปรับซื้อถึงบ้าน

**เข้าเดโม:** หน้า login → ปุ่ม **"ผู้ซื้อ / คนขับ"**

1. **บัญชี** → ตั้ง **ราคารับซื้อของฉัน** (หรือใช้ราคากลาง)
2. **จัดตารางรับของ** — เปิดรอบ (วัน + โซน + จำนวนคิว) ให้ผู้ขายจอง
3. **งานของฉัน** — เห็น **งานใกล้ฉัน (รัศมี 30 กม.)** เรียงจากใกล้สุด + งานที่ถูกจองมา
4. เปิดงาน → **รับงาน / คอนเฟิร์ม** → **เริ่มเดินทางไปรับ** → ชั่งของ → **ปิดงาน & ออกบิล**
   - เห็นสรุป: ราคาของ − ค่าบริการ 5% = **ยอดจ่ายผู้ขาย** → บันทึก (บิลเข้าระบบร้านอัตโนมัติ)

---

## 🏪 3. เจ้าของร้านรับซื้อ (Shop Back-office) — เว็บเดสก์ท็อป

**เข้าเดโม:** login เป็นผู้ซื้อ → แท็บ **บัญชี** → การ์ด **"ระบบร้านรับซื้อ"** → `/shop` *(เปิดจอกว้าง)*

1. **ภาพรวม** — รับซื้อวันนี้/รวม · จ่ายผู้ขาย · กราฟ 7 วัน · บิลล่าสุด
2. **บิลรับซื้อ → สร้างบิล**
   - **walk-in** (ลูกค้าหน้าร้าน กรอกเอง) หรือ **จากงานในแอป** (เลือกงานที่คอนเฟิร์ม)
   - ชั่ง + ใส่ราคา → หักค่าบริการ → **ยอดจ่ายสุทธิ** → บันทึก → **พิมพ์บิล/PDF**
3. **บัญชี** — รายรับ(ค่าบริการ) − รายจ่าย (น้ำมัน/แรง/เช่า) = กำไร · **ส่งออก CSV**
4. **ราคา/วัสดุ** — ตั้งราคารับซื้อของร้าน

---

## 🛡️ 4. เจ้าของระบบ (Admin) — เว็บเดสก์ท็อป

**เข้าเดโม:** หน้า login → ลิงก์ **"เข้าระบบผู้ดูแลระบบ (Admin) →"** → `/admin`

1. **ภาพรวม** — **GMV รวมทุกผู้ซื้อ** · **ค่าคอมแพลตฟอร์ม 2%** · GMV รายเดือน · Top ผู้ซื้อ
2. **ผู้ซื้อ** — รายชื่อทุกคน + GMV/บิล + ค้นหา + **อนุมัติ/ระงับบัญชี** (มีผลบังคับจริง)
3. **รางวัล** — ตั้งรางวัล · ดูผู้มีสิทธิ์ทุกคน · **สุ่มผู้โชคดี → ประกาศ → แจ้ง LINE ผู้ขายทุกคน** (สุ่มยุติธรรมจากสิทธิ์จริง)
4. **ราคากลาง** — แก้ราคาที่ผู้ขายเห็น (มีผลทั้งระบบ)
